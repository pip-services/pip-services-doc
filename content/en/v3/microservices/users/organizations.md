Keeps list of working organizations.

- Server implementations: [NodeJS](https://github.com/pip-services-users/pip-services-organizations-node)
- Communication protocols: Direct, Commandable HTTP, Commandable GRPC, GRPC
- Persistence: Memory, Flat Files, MongoDB, Couchbase
- Deployment options: Monolith, Process, [Docker](https://hub.docker.com/u/pipdevs), AWS Lambda
- Client implementations: [NodeJS](https://github.com/pip-services-users/pip-clients-organizations-node), [Golang](https://github.com/pip-services-users/pip-clients-organizations-go)
