### Intallation procedure

PIP.Services needs to be installed per module. The general form is:

```bash
npm i module-name
```

### Module information:


#### AWS module

```bash
npm i pip-services3-aws-nodex --save
```
[GitHub repo](https://github.com/pip-services3-nodex/pip-services3-aws-nodex)
