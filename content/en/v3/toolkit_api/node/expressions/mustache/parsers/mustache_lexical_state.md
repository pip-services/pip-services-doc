### Description

The MustacheLexicalState enum defines different states in Mustache lexical analysis.


### Enumeration members

The codes are as follows:

- **Value** = 0
- **Operator1** = 1
- **Operator2** = 2
- **Variable** = 3
- **Comment** = 4
- **Closure** = 5
