### Description

The INamed interface is used to define data objects that have a human-readable name.

### Properties


#### Name
The object's humand-readable name.
> string Name { get; set; }

### Examples
```cs
public class MyData: IStringIdentifiable, INamed 
{
    string id {get; set;}
    string name {get; set;}
    string field1;
    int field2;
    ...
}
```
