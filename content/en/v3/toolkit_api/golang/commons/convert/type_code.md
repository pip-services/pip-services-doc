### Description
 
The TypeCode class contains the codes for data types that are used by the class [TypeConverter](../type_converter). 

### Enumeration members

The codes are as follows:

- **Unknown** = iota
- **String**
- **Boolean**
- **Integer**
- **Long**
- **Float**
- **Double**
- **DateTime**
- **Duration**
- **Object**
- **Enum**
- **Array**
- **Map**
