### Description

The VariantType enum defines supported types of variant values.


### Enumeration members

The codes are as follows:

- **Null** = iota
- **Integer**
- **Long**
- **Float**
- **Double**
- **String**
- **Boolean**
- **DateTime**
- **TimeSpan**
- **Object**
- **Array**
