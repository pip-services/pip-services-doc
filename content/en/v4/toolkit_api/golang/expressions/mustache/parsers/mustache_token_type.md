### Description

The MustacheTokenType enum defines different types of Mustache tokens.


### Enumeration members

The codes are as follows:

- **Unknown** = iota
- **Value**
- **Variable**
- **EscapedVariable**
- **Section**
- **InvertedSection**
- **SectionEnd**
- **Partial**
- **Comment**
