### Description

The VariantType enum defines supported types of variant values.


### Enumeration members

The codes are as follows:

- **Null** = 0
- **Integer** = 1
- **Long** = 2
- **Float** = 3
- **Double** = 4
- **String** = 5
- **Boolean** = 6
- **DateTime** = 7
- **TimeSpan** = 8
- **Object** = 9
- **Array** = 10
